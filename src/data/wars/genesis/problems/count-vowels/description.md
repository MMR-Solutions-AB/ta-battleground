# Count vowels

Skriv en funktion **countVowels(s)** som tar emot en sträng s. Funktionen ska returnera antalet vokaler i strängen s. Vokalerna i det engelska alfabetet är **"a, e, i, o ,u"**

## Exempel 1

**_Input_**

```bash
s = "javascript"
```

**_Output_**

```bash
3
```

## Exempel 2

**_Input_**

```bash
s = "techover"
```

**_Output_**

```bash
2
```
