# Is positive

Skriv en funktion **isPositive(n)** som returnerar **true** om **n** är positivt, annars **false**.

Denna uppgift är bra tillfälle att träna på **logiska operators** som **>** eller **<** och **booleanska** värden **(true eller false)**

## Exempel 1

**_Input_**

```js
n = 4;
```

**_Output_**

```bash
true
```

## Exempel 2

**_Input_**

```js
n = -5;
```

**_Output_**

```bash
false
```
