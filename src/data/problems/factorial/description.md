# Factorial

Skriv en funktion **factorial(n)** som returnerar **n!**, det vill säga **n** fakultet. **n!** definieras som produkten av alla heltal från 1 till **n**.

## Exempel 1

**_Input_**

```js
n = 3;
```

**_Output_**

```bash
6
```

**_Förklaring_**

**3!** blir **3 \* 2 \* 1** = **6**

## Exempel 2

**_Input_**

```js
n = 5;
```

**_Output_**

```bash
120
```

**_Förklaring_**

**5!** blir **5 \* 4 \* 3 \* 2 \* 1** = **120**
