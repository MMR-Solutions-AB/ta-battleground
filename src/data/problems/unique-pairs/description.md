# Unique pairs

Skriv en funktion **uniquePairs(n)** som genererar en lista med alla unika par av heltal **(i, j)** där **i** och **j** är heltal i intervallet **[0, n]** och **i <= j**.

## Exempel 1

**_Input_**

```bash
n = 2
```

**_Output_**

```bash
[[0, 0], [0, 1], [0, 2], [1, 1], [1, 2], [2, 2]]
```

## Exempel 2

**_Input_**

```bash
n = 3
```

**_Output_**

```bash
[[0, 0], [0, 1], [0, 2], [0, 3], [1, 1], [1, 2], [1, 3], [2, 2], [2, 3], [3, 3]]
```
