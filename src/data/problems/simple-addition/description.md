# Simple addition

Givet två nummer, **a** och **b**, beräknade summan av **a + b** och returnera det ur funktionen **simpleAddition**. Alltså addera **a** och **b** och returnera ditt svar

## Exempel

### Input

```bash
a = 1
b = 2
```

### Output

```bash
3
```

### Förklaring

**1 + 2** = **3**
