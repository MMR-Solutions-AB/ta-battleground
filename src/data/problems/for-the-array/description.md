# For the array

Skriv en funktion **forTheArray(n)** som tar ett nummer **n** och returnerar en array med alla heltal från **1** till och med **n**.

Denna uppgift är bra tillfälle att träna på **for loops** och **arrays**

## Exempel 1

**_Input_**

```bash
n = 3
```

**_Output_**

```bash
[1, 2, 3]
```

## Exempel 2

**_Input_**

```bash
n = 5
```

**_Output_**

```bash
[1, 2, 3, 4, 5]
```
