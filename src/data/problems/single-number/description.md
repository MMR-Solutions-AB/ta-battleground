# Single number

Skriv en funktion **singleNumber(nums)** som tar emot en array av positiva heltal där varje tal förekommer två gånger förutom ett number. Funktionen ska returnera det numret som enbart förekommer en gång

## Exempel 1

**_Input_**

```bash
nums = [2, 2, 1]
```

**_Output_**

```bash
1
```

## Exempel 2

**_Input_**

```bash
nums = [4, 1, 2, 1, 2]
```

**_Output_**

```bash
4
```
