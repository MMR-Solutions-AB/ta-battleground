# Simple multiplication

Givet två nummer, **a** och **b**, beräknade produkten av **a \* b** och returnera det ur funktionen **simpleMultiplication**. Alltså multiplicera **a** och **b** och returnera ditt svar

## Exempel

### Input

```bash
a = 1
b = 2
```

### Output

```bash
2
```

### Förklaring

**1 \* 2** = **2**
