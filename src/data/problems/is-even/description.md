# Is even

Skriv en funktion **isEven(n)** som returnerar **true** om n är jämnt, annars **false**.

Denna uppgift är bra tillfälle att träna på **modulo-operatorn (%)**

## Exempel 1

**_Input_**

```js
n = 4;
```

**_Output_**

```bash
true
```

## Exempel 2

**_Input_**

```js
n = 5;
```

**_Output_**

```bash
false
```
