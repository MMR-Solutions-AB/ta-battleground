# Uppercase array

Skriv en funktion **uppercaseArray(arr)** som tar emot en array av strängar. Funktionen ska returnera en samma array fast med varje sträng med stora bokstäver

## Exempel 1

**_Input_**

```bash
arr = ["hello", "world"]
```

**_Output_**

```bash
["HELLO", "WORLD"]
```

## Exempel 2

**_Input_**

```bash
arr = ["ronALdo", "tHe", "GOAT"]
```

**_Output_**

```bash
["RONALDO", "THE", "GOAT"]
```
