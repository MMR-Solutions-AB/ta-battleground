# Simple division

Givet två nummer, **a** och **b**, beräknade kvoten av **a / b** och returnera det ur funktionen **simpleDivision**. Alltså dividera **a** med **b** och returnera ditt svar

## Exempel

### Input

```bash
a = 10
b = 5
```

### Output

```bash
2
```

### Förklaring

**10 / 5** = **2**
